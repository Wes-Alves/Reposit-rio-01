Comandos GIT:

- git status: serve para visualizar mudanças nos arquivos.

- git diff: serve para visualizar as mudanças que foram feitas nos arquivos.

- git add -A: serve para adicionar ao repositório.

- git commit -m: serve para criar o Commit.

- git branch: serve para listar todos os Branchs.

- git branch "Nome do branch": serve para criar um novo branch.

- git checkout "Nome do branch": serve para mudar de branch.
