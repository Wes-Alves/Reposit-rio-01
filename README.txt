Comandos GIT:

- git status: serve para visualizar mudanças nos arquivos.

- git diff: serve para visualizar as mudanças que foram feitas nos arquivos.

- git add -A: serve para adicionar ao repositório.

- git commit -m "Nome do commit": serve para criar o Commit.

- git branch: serve para listar todos os Branchs.

- git branch "Nome do branch": serve para criar um novo branch.

- git checkout "Nome do branch": serve para mudar de branch.

- git commit -am "Nome do commit" ou git add -A && git commit -m "Nome do commit": serve para adicionar o arquivo ao repositório e criar o commit ao mesmo tempo.

- git reset --hard (commit que deseja voltar): serve para voltar para commits anteriores.

- git remote -v: serve para identificar a url remota do repositório.

- git remote rm origin: serve para apagar repositórios.

- git remote add origin (url do repositório): serve para conectar com o repositório remoto:

- git pull origin master: serve para sincronizar os repositórios.
